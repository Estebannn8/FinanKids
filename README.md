# FinanKids

**FinanKids** es una aplicación móvil educativa diseñada para enseñar conceptos básicos de finanzas, ahorro y gestión de dinero a niños y jóvenes de una forma divertida e interactiva.  
Este proyecto es desarrollado como parte de un proyecto universitario por un equipo de tres estudiantes.

## Características principales (propuestas)

- Lecciones interactivas sobre ahorro, ingresos y gastos.
- Música y sonidos ambientales para mantener la atención.
- Interfaz colorida y amigable para niños.
- Seguimiento de metas de ahorro (simulado).
- Actividades y preguntas para reforzar el aprendizaje.

> *Nota: la lista completa de funcionalidades se encuentra en desarrollo. Puedes revisar los commits recientes para ver el avance.*

## Tecnologías utilizadas

- **Lenguaje:** Kotlin 100%
- **Entorno:** Android (con Gradle)
- **IDE recomendado:** Android Studio
- **Control de versiones:** Git + GitHub

## ¿Cómo probar el proyecto?

1. **Clona el repositorio**
   
   ```bash
   git clone https://github.com/Estebannn8/FinanKids.git

2. Abre el proyecto en Android Studio.

3. Sincroniza las dependencias de Gradle.

4. Ejecuta la app en un emulador o dispositivo físico Android.

Requisitos: Android Studio Ladybug o superior, JDK 11 o superior.
